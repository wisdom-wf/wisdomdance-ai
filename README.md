# WisdomDance.ai

王凡的AI个人主页 | AI+政务数字化专家

## 项目简介

WisdomDance.ai（智慧跳动）是王凡的OpenClaw个人主页，展示AI+政务数字化服务能力。

## 技术栈

- Next.js 14 + React 18
- Tailwind CSS
- shadcn/ui
- TypeScript
